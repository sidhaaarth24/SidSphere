# SidSphere

A modern, responsive developer portfolio website built with HTML, CSS, JavaScript.

![Status](https://img.shields.io/badge/status-active-success)
![Frontend](https://img.shields.io/badge/frontend-HTML%20%7C%20CSS%20%7C%20JavaScript-blue)

## 🚀 Overview

SidSphere is a personal portfolio website designed to showcase skills, projects, achievements, certifications, and professional information through a clean coder-inspired interface.

The project features an interactive terminal, theme switching, animated UI elements, and a fully responsive design.

Preview:
https://sidhaaarth24.github.io/SidSphere

---

## ✨ Features

- Responsive design for desktop and mobile devices
- Dark / Light mode toggle
- Interactive terminal interface
- Typing text animations
- Project showcase section
- Education & experience timeline
- Certificates and awards gallery
- Contact form
- Social media integration
- Smooth animations and transitions

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/sidhaaarth24/SidSphere.git
```

Move into the project directory:

```bash
cd SidSphere
```

Run a local server:

```bash
python -m http.server 8000
```

Or simply open `index.html` in your browser.
